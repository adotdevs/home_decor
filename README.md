# Luxe Home Decor Ideas\n\nProduction-style Next.js content platform for premium home decor publishing.\n\n## Stack\n- Next.js App Router + TypeScript\n- Tailwind CSS + shadcn/ui + Framer Motion\n- MongoDB + Mongoose\n- JWT admin auth (single superadmin bootstrap)\n- REST route handlers\n\n## Quick start\n1. Copy `.env.example` to `.env.local`\n2. Install dependencies: `npm install`\n3. Seed content: `npm run seed`\n4. Start dev server: `npm run dev`\n\n## Scripts\n- `npm run dev`\n- `npm run lint`\n- `npm run build`\n- `npm run seed`\n\n## Deployment\n- Deploy on Vercel\n- Set environment variables from `.env.example`\n- Ensure MongoDB Atlas network access for deployment regions
